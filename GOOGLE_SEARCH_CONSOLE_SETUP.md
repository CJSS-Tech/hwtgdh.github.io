# Google Search Console 设置指南

## 📋 完整操作步骤

### ✅ 第一步：访问 Google Search Console

1. **打开浏览器**，访问：
   ```
   https://search.google.com/search-console
   ```

2. **登录** 你的 Google 账号（与 Google Analytics 使用同一个账号即可）

---

### ✅ 第二步：添加网站资源

1. 点击左上角的 **"添加资源"** 或 **"Add property"**

2. 选择资源类型：
   ```
   ☑️ 网址前缀 (URL prefix)  ← 选择这个！
   ```

3. 输入网站 URL：
   ```
   https://cjss-tech.github.io/hwtg/
   ```
   
4. 点击 **"继续"** 或 **"Continue"**

---

### ✅ 第三步：验证网站所有权

Google 会要求你验证网站所有权。有多种验证方法：

#### 🎯 推荐方法一：HTML 标签验证（最简单）

1. **Google Search Console 会显示类似这样的代码：**
   ```html
   <meta name="google-site-verification" content="xxxxxxxxxxxxxx" />
   ```

2. **复制这个完整的 meta 标签**

3. **告诉我这个标签**，我会帮你添加到 `index.html` 的 `<head>` 部分

4. 等待 GitHub Pages 部署（3-5分钟）

5. **返回 Google Search Console**，点击 **"验证"** 按钮

#### 🎯 方法二：HTML 文件验证

1. Google 会提供一个文件（如 `googlexxxxxxxxxxxx.html`）
2. **告诉我文件名**，我会帮你创建这个文件
3. 提交到 GitHub
4. 等待部署后点击 **"验证"**

#### 🎯 方法三：通过 Google Analytics 验证（推荐！✨）

**这个最简单！** 因为你已经添加了 Google Analytics：

1. 在验证方法中选择 **"Google Analytics"**
2. 确认使用的是同一个 Google 账号
3. 点击 **"验证"** - 应该会**自动通过**！✅

---

### ✅ 第四步：提交 Sitemap

**验证成功后：**

1. 在左侧菜单找到 **"站点地图"** 或 **"Sitemaps"**

2. 在 **"添加新的站点地图"** 输入框中输入：
   ```
   sitemap.xml
   ```
   
   ⚠️ **注意**：只需输入 `sitemap.xml`，不需要完整 URL！
   
   因为你已经添加了 `https://cjss-tech.github.io/hwtg/`，所以系统会自动拼接为：
   ```
   https://cjss-tech.github.io/hwtg/sitemap.xml
   ```

3. 点击 **"提交"** 或 **"Submit"**

4. **状态显示**：
   - 🟡 **"待处理"** - 正在处理，正常现象
   - 🟢 **"成功"** - 已被 Google 读取（可能需要几小时到几天）
   - 🔴 **"错误"** - 有问题（告诉我错误信息，我帮你解决）

## 🚨 常见问题

### Q1: 找不到 "添加资源" 按钮？

**解决方法**：
- 点击左上角的 **"搜索资源"** 下拉菜单
- 在下拉菜单底部找到 **"添加资源"** 或 **"+ 添加资源"**

### Q2: 验证一直不通过？

**可能原因**：
1. GitHub Pages 还没部署完成 → **等待 10 分钟后重试**
2. Meta 标签位置不对 → **确保在 `<head>` 标签内**
3. 使用了错误的 Google 账号 → **切换到 GitHub 关联的账号**

### Q3: Sitemap 提交后显示错误？

**检查清单**：
- ✅ 确认 sitemap.xml 文件已推送到 GitHub
- ✅ 访问 `https://cjss-tech.github.io/hwtg/sitemap.xml` 能打开
- ✅ XML 格式正确（已验证 ✅）
